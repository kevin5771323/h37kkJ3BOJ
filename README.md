# 项目简介

## 核心功能
- 基于YOLOv8的遥感建筑实例分割检测
- 支持14000张包含'Building'类别的标注数据集
- 提供完整源码、训练好的模型及训练结果
- 包含项目运行教程和使用说明
- 支持图片、视频及摄像头进行检测
- 支持自定义置信度和IOU阈值
- 提供批量检测功能，可查看所有检测结果
- 支持检测结果保存
- 界面实时显示目标位置、目标总数和置信度信息

## 适用场景
- 遥感图像中的建筑检测与分析
- 城市规划与地理信息系统
- 遥感图像处理与自动化识别
- 建筑物实例分割研究
- 计算机视觉教学与实验

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9技术栈
- 集成OpenCV、PyQt5、Ultralytics和PyTorch等框架
- 提供图形用户界面，操作简便直观
- 模块化设计，支持多种输入源和参数调整
